# รูปที่ต้องใช้ — วาง Path ตามนี้แล้วรูปจะขึ้นในเว็บทันที

**สถานะล่าสุด**: ได้รับแล้ว ✓ hero/teacher-hero, classroom/explaining, classroom/moment, students/success-01, students/success-02, articles ×3 (เป็นไฟล์ .png), mascot/sticker-01 — เหลือ posts/*.jpg (6 ไฟล์) ที่ยังรออยู่

เว็บถูก wire ไว้ให้ดึงรูปจาก path ด้านล่างอัตโนมัติ — ถ้ายังไม่มีไฟล์ ระบบจะโชว์กรอบ placeholder พร้อม label แทน (ไม่พัง) พอวางไฟล์ชื่อตรงกันแล้ว รูปจะขึ้นแทนที่ทันทีโดยไม่ต้องแก้โค้ด

รายการฉบับเต็มพร้อมขนาดพิกเซล คุยกันในแชทแล้ว — ไฟล์นี้เป็นสำเนาไว้อ้างอิงตอนเซฟไฟล์จริง
